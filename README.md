# AI-Enhanced Code Review Assistant

A thesis project combining web development with AI research to evaluate 
whether AI-assisted code review is more accurate, faster, and more 
trustworthy than manual code review.

## Project Overview

- **Team:** Hasitha Hiththatiyage, Nipuni Kodikara
- **Duration:** 12 weeks
- **Tech Stack:** React, Node.js, Supabase, OpenAI
- **Focus:** JavaScript/React code review

## Folder Structure
├── frontend/          React application
├── backend/           Express server
├── database/          SQL schemas and migrations
└── docs/              Documentation

## Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn
- Supabase account
- OpenAI API key

### Setup

1. Clone repository
2. Create `.env` file (copy from `.env.example`)
3. Add your API keys
4. Follow setup instructions in frontend/ and backend/ folders
